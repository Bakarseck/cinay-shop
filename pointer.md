## Pointer

C'est l'adresse d'une variable.

Pour déclarer un variable, on met:

```c
```

Pour déclarer un pointeur, on met:

```c
     int a;
     int *ptr;

     ptr = &a;

     printf("%d", a);
     printf("%d", *ptr);

     printf("%d", ptr);
     printf("%d", &a);
```

Pour acceder à la variable, contenu dans l'adresse, on met *devant le pointeur.

```c
     printf();
```
